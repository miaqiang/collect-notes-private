### 强制英文换行
```
word-break: break-all;
word-wrap: break-word
```

### 显示一行，超过长度显示省略号

```
overflow: hidden;
text-overflow: ellipsis;
white-space: nowrap;
```

### 显示三角符号

方向向右

```
border-top: 8px solid transparent;
border-left: 10px solid rgb(255, 255, 255);
border-bottom: 8px solid transparent;
```
